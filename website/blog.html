FastText 工具包

FastText 是 Facebook 开源的自然语言处理工具包，核心设计理念是在保持较高精度的前提下，实现极快的训练和预测速度。主要解决两大问题：词向量训练和文本分类（二分类/单标签多分类/多标签多分类）。

两大加速技术：① 层次 Softmax——使用哈夫曼树组织类别，将复杂的多分类问题转化为一系列二分类问题，利用哈夫曼编码（左分支 0、右分支 1）表示从根节点到叶子节点的路径，通过极大似然估计计算概率并更新参数；② 负采样——训练词向量时只更新正样本和随机采样的少量负样本的权重，无需更新整个巨大的权重矩阵，大幅降低计算量。

虽然模型结构简单，但通过 n-gram 特征弥补了无法捕捉词序的问题，让模型理解局部词语顺序，显著提升分类精度。

项目实战流程：① 数据准备——格式化数据，标签前缀为 __label__；② 模型训练——API 为 fasttext.train_supervised；③ 模型调优——调整训练轮数、学习率、n-gram、损失函数，支持自动调参；④ 评估与预测——model.test 评估准确率，model.predict 进行预测；⑤ 模型保存与加载——save_model / load_model。

自然语言处理 NLP · 文本预处理

NLP（自然语言处理）是让计算机理解、生成、处理人类语言的技术，文本预处理是 NLP 的第一步，目的是将原始文本变成干净、规范、适合模型输入的格式。

文本预处理的主要环节：数据层面——了解数据分布、特征分布、标签分布并可视化；特征层面——标准化/归一化、增加 n-gram 特征；文本处理基本方法——分词、词性标注、命名实体识别；文本张量表示——将文字转换为数字序列。

文本处理三大方法：① 分词——jieba 是目前最流行的中文分词库，提供精确模式、全模式和搜索引擎模式，支持自定义词典，对未登录词借助 HMM 模型做序列标注，「词典 + 统计 + HMM」的混合策略使其速度快、准确率高；② 词性标注（POS）——给每个词赋予语法类别（动词 v、名词 n 等），是理解句子结构的基础；③ 命名实体识别（NER）——从文本中找出专有名词并分类（人名、地名、机构名、时间、日期等）。

文本张量表示三大方法：① One-hot 独热编码——词数即维度，仅当前词位置为 1、其余为 0，简单但维度爆炸；② Word2Vec——语义相似的词具有相似的上下文，包含 CBOW（上下文预测中心词，擅长高频词、训练快）和 Skip-gram（中心词预测上下文，擅长低频词、对罕见词友好），底层核心都是巧妙地应用向量矩阵乘法；③ Word Embedding 词嵌入——将词汇映射到低维稠密实向量空间，广义上包含 Word2Vec 和词嵌入层 torch.nn.Embedding，狭义上专指词嵌入层。

FastText 是 Facebook 2016 年开源的工具包，专注效率与实用性，核心功能：训练词向量 + 文本分类。

决策树 · 集成学习 · 随机森林

决策树是一种既可分类也可回归的树形结构模型，通过一系列判断规则进行预测。构建过程有三要素：特征选择、决策树生成和决策树剪枝。特征选择算法主要有 ID3、C4.5 和 CART，最常用的是 CART 决策树，其指标为基尼系数，优点是计算快、支持回归和分类。决策树剪枝用于缓解过拟合，分为预剪枝（计算效率高、训练快）和后剪枝（保留更多有效结构、泛化性能更好）。

集成学习是通过组合多个弱学习器构建强学习器的机器学习范式，主要方法有 Bagging 和 Boosting。Bagging（代表：随机森林）采用有放回抽样、并行训练、平权投票的方式；Boosting（代表：AdaBoost、XGBoost）使用全部样本、串行训练、加权投票，每次重点关注前一学习器不足的地方。

卷积神经网络 CNN · 循环神经网络 RNN

CNN 是处理图像的专用深度学习模型，核心思想是局部连接、参数共享和层次化特征提取。图像在 NumPy 中表示为 (H, W, C)，PyTorch 中为 (C, H, W)。CNN 流程：输入图像 → 多层卷积（每个卷积核做点积运算输出特征图）→ 池化降维 → 全连接分类。Padding 补零防止边缘信息丢失，Stride 步长控制滑动距离，特征图公式为 N = ⌊(W−F+2P)/S⌋ + 1。API：nn.Conv2d(in_channels, out_channels, kernel_size, stride, padding)，池化层 MaxPool2d / AvgPool2d 仅在 H、W 维度降维，不改变通道数。

RNN 是处理序列数据的网络模型，核心是具有"记忆"功能的循环结构。每个时间步同时接收当前输入（词向量）和上一时间步的隐藏状态，通过共享的循环单元计算新隐藏状态并产生输出。流程：分词（jieba）→ 词映射下标 → nn.Embedding 转为稠密向量 → RNN 层 → 全连接 + Softmax 输出概率。简单 RNN 易梯度消失，实际常用 LSTM 或 GRU。

梯度下降的优化方式 & 正则化

梯度下降算法演进路线：标准梯度下降（全量数据，慢）→ SGD（随机小批量，快但震荡）→ 动量法 Momentum（积累历史梯度，抑制震荡加速收敛）→ AdaGrad（自适应学习率，但后期衰减至零）→ RMSprop（指数加权平均替代累加，解决早停）→ Adam（动量法 + RMSprop，兼顾方向和尺度，默认优化器）→ AdamW（权重衰减与自适应解耦，Transformer 中更优）。

正则化是防止过拟合的技术手段。三大类策略：范数正则化——L2 正则化让权重变小但不为零（模型平滑），L1 正则化让部分权重精确为零（特征选择、模型稀疏）；Dropout——训练时随机置零神经元输出，相当于每次训练子网络，迫使神经元独立学习；Batch Normalization——对每层输入做标准化（均值 0 方差 1），再学缩放 γ 和平移 β 恢复表达能力。

损失函数 & 网络模型优化方法

损失函数根据任务类型分为分类和回归两类。分类损失：多分类交叉熵 CrossEntropyLoss 底层自动 Softmax 转概率，直接传预测分数即可；二分类交叉熵 BCELoss 需手动 Sigmoid 转概率后再传入。回归损失：MAE（L1Loss）对大误差不放大、梯度恒定，但 0 点不可导易震荡；MSE（MSELoss）处处可导但放大异常样本误差易梯度爆炸；Smooth L1（Huber Loss）结合两者优点，[-1,1] 内导数减小、之外导数为 ±1，对异常鲁棒且不易跳过最低点。

优化核心公式：新参数 = 旧参数 − 学习率 × 梯度。两个可调方向：梯度（SGD → 动量法优化）和学习率（AdaGrad → RMSprop → Adam）。指数加权平均是关键思想——给近期数据更高权重、老数据指数衰减：V_t = β·V_{t-1} + (1−β)·θ_t。

神经网络核心概念：激活函数、权重与偏置

神经网络由多个神经元按层连接组成，训练时数据前向传播得到预测值，通过损失函数衡量误差，再反向传播计算梯度并更新参数，如此反复迭代。七大核心概念：神经元、层数、权重与偏置、激活函数、前向传播、损失函数、反向传播。

四大激活函数对比：Sigmoid——值域 (0,1)，x∈[-6,6] 明显，二分类输出层，非 0 中心化且梯度消失严重（多层连乘 ≈0.25ⁿ 趋零）；Tanh——值域 (-1,1)，x∈[-3,3] 明显，浅隐藏层，0 中心化缓解梯度消失；ReLU——值域 [0,+∞)，负样本忽略正样本直通，深隐藏层，缓解过拟合；Softmax——概率和为 1，多分类输出层。口诀：浅层 Tanh · 深层 ReLU · 二分类输出 Sigmoid · 多分类输出 Softmax。

参数初始化是模型训练的起跑线。Kaiming 初始化只考虑输入节点（fan_in），适用于 ReLU 及其变体；Xavier 初始化同时考虑输入和输出（fan_in + fan_out），适用于 Sigmoid/Tanh；如果用 ReLU 配合 Xavier，梯度方差会减半。

PyTorch 张量操作 & 自动微分

张量操作常用 API：索引操作格式为 张量[0轴, 1轴, 2轴]；形状操作——reshape（可改连续和不连续张量）vs view（仅改连续张量，用 is_contiguous() 判断）；squeeze 降维（一次可降多个）vs unsqueeze 升维（一次一个）；transpose 交换两个维度 vs permute 交换多个维度；拼接——cat（维数不变，拼接维外形状需一致）vs stack（维数 +1，所有维度需完全相同）。

自动微分流程：① 设置 requires_grad=True 标记参数 w、b 需跟踪；② 执行前向传播（矩阵乘法 + 损失计算），PyTorch 自动构建计算图；③ 调用 .backward() 触发反向传播，自动计算梯度存入 .grad 属性；④ 用公式 新参数 = 旧参数 − 学习率 × 梯度 更新参数完成梯度下降优化。整个机制动态构建计算图、自动求解梯度。

深度学习概述

深度学习是机器学习的重要分支，核心优势是自动从原始数据中逐层学习层级特征，无需人工设计特征。依赖大规模数据和 GPU/TPU 并行加速，可解释性弱但在图像、语音、NLP 及大模型领域表现卓越。

AI 发展关键节点：1956 年麦卡锡提出 AI 概念 → 2012 年 AlexNet 引爆深度学习 → 2015 年 ResNet152 错误率降至 3% → 2017 年 Transformer 诞生 → 2022 年 ChatGPT 问世 → 2025 年 DeepSeek 开启智能体时代。

训练核心：前向传播做预测（输入→逐层计算→输出），反向传播学教训（误差→倒推梯度→更新参数），两者反复迭代。常见算法：ANN、CNN、RNN、Transformer。PyTorch 是最流行的深度学习框架，数据以张量存储，提供丰富的创建、转换、运算、索引、形状操作和拼接 API。

逻辑回归（分类）· K-Means 聚类

逻辑回归是一种擅长二分类的有监督学习算法：先将线性回归输出送入 Sigmoid 函数压缩到 (0,1)，解释为正类概率，再设定阈值分类。优化方面，从最大似然估计出发，对似然函数取负对数推导出交叉熵损失函数——收敛稳定、适配梯度下降。评估指标：准确率、精确率（TP/(TP+FP)）、召回率（TP/(TP+FN)）、F1 分数（调和平均）。混淆矩阵是核心评估工具。

K-Means 是无监督聚类算法，目标是将数据划分为 K 个互不重叠的簇。核心流程：随机初始化 K 个质心 → 反复执行"分配（每个样本→最近质心）→ 更新（均值→新质心）"直到收敛。K 值选择参考三类指标：SSE（肘部法则）、轮廓系数 SC 和 CH 指数——后两者越高越好，最高点对应最优 K。

线性回归 & 梯度下降

线性回归假设标签与特征之间存在线性关系，目标是拟合一条直线（或超平面）使误差最小。损失函数常用最小二乘法（本质是 MSE），使用平方的好处：正负误差不抵消、放大较大误差让模型关注偏离严重的样本。

两种求解方法：正规方程——对 MSE 求偏导令其为零，一步到位解出最优参数，简单直接但计算量大，矩阵不可逆时无解，仅适合小数据；梯度下降——迭代更新逼近最优解，公式为 新参数 = 旧参数 − 学习率 × 梯度，需手动设学习率和迭代次数，但灵活通用，适合各种 AI 场景。sklearn 对应 API：LinearRegression（正规方程）和 SGDRegressor（梯度下降）。评估三指标：MAE、MSE、RMSE。

KNN 距离度量 · 特征预处理 · 交叉验证

四种距离度量：欧式距离——各维度差的平方和再开方（最常用）；曼哈顿距离——各维度差的绝对值相加；切比雪夫距离——各维度差的最大值；闵式距离——前三种的统一公式：各维度差的绝对值的 p 次方相加再开 p 次方。

特征预处理：为了让不同尺度的特征公平参与计算，需做标准化或归一化。归一化 = (原始值 − 最小值) / (最大值 − 最小值)；标准化 = (原始值 − 平均值) / 标准差。配合 交叉验证（K 折 CV：均分 K 份，每份轮流做验证集）和 网格搜索（寻找最优超参数）来提升模型效果。

机器学习概述 · KNN 算法入门

核心概念关系：AI ⊃ ML ⊃ DL。机器学习是实现人工智能的主要途径，深度学习是机器学习的一种方法（多层神经网络）。AI 发展三要素：数据、算法、算力（CPU/GPU/TPU）。数据由样本（一行记录）、特征/属性（描述维度）和标签（预测目标）组成。

算法分类：根据标签情况分为四类——有监督学习（有标签，分为离散分类和连续回归）、无监督学习（无标签，典型任务是聚类）、半监督学习（部分标签，降低标注成本）、强化学习（智能体与环境互动获取最大累积奖励）。

建模五步流程：获取数据 → 数据预处理（清洗异常/缺失值）→ 特征工程 → 模型训练 → 模型评估。KNN 算法思想是"近朱者赤"：计算未知样本到所有已知样本的欧式距离 → 取最近的 K 个邻居 → 分类投票 / 回归取平均值。

Dify 平台 · 微调 · RAG 核心流程

多智能体架构 & MCP/A2A 协议

多智能体架构由父 Agent（路由 Agent）和多个子 Agent（SubAgent）组成。路由 Agent 负责接收初始任务、进行意图识别和任务分解，每个子 Agent 根据各自功能完成子任务。

在智能体世界中，A2A（Agent-to-Agent）和 MCP（Model Context Protocol）解决两个核心问题：A2A 解决智能体之间如何协作的问题，是不同智能体间的通用协作语言；MCP 解决智能体如何运用工具的问题，是智能体和外部世界的"万能插头"——一套让 AI 模型用统一方式安全连接和使用外部工具与数据源的标准。
