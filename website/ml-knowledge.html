AI概述

人工智能（AI）是计算机科学的一个分支，旨在创建能够模拟人类智能的系统。机器学习是实现人工智能的核心方法之一。

核心观点

人工智能 > 机器学习 > 深度学习。三者是包含关系：深度学习是机器学习的一个子集，机器学习是人工智能的一个子集。

三大流派

  • 符号主义：基于逻辑推理，用符号表示知识，如专家系统
  • 连接主义：基于神经网络，模拟人脑神经元连接，当前主流方向
  • 行为主义：基于"感知-行动"循环，如强化学习

机器学习的本质

  • 从数据中自动学习规律和模式
  • 利用学到的规律对未知数据进行预测或决策
  • 核心三要素：数据 + 模型 + 优化算法
  • 目标：最小化预测值与真实值之间的误差

AI / ML / DL 关系

理清人工智能、机器学习、深度学习三者之间的层次关系，是建立完整知识体系的第一步。

一句话总结

AI 是目标，ML 是实现路径，DL 是 ML 中最强大的技术分支。

概念定义特点
人工智能 AI 让机器展现智能行为的科学 范围最广，包含推理、规划、学习、感知等
机器学习 ML 从数据中自动学习规律 不需要显式编程，依赖数据和统计方法
深度学习 DL 使用深层神经网络进行学习 自动提取特征，适合图像、语音、NLP 等复杂任务

发展脉络

  • 1956 达特茅斯会议提出 AI 概念
  • 1980s 机器学习兴起（决策树、SVM 等）
  • 2006 Hinton 提出深度信念网络，深度学习复兴
  • 2012 AlexNet 在 ImageNet 夺冠，CNN 时代开启
  • 2017 Transformer 提出，大模型时代来临

机器学习分类

根据训练数据是否有标签、学习方式的不同，机器学习可分为三大类。

类型数据标签核心任务典型算法
监督学习 有标签 (X, y) 分类、回归 线性回归、决策树、SVM
无监督学习 无标签 (X) 聚类、降维、关联规则 K-Means、PCA、DBSCAN
半监督/强化学习 部分标签 / 奖励信号 序列决策 Q-Learning、Deep Q-Network

如何选择？

  • 有标签 + 预测数值 → 回归（监督学习）
  • 有标签 + 预测类别 → 分类（监督学习）
  • 无标签 + 发现分组 → 聚类（无监督学习）
  • 无标签 + 降低维度 → 降维（无监督学习）

核心术语

掌握以下术语是理解机器学习的前提，这些概念贯穿所有算法的学习过程。

基础概念

  • 特征（Feature）：输入变量，描述样本的属性，记为 X
  • 标签（Label）：输出变量，需要预测的目标值，记为 y
  • 样本（Sample）：一条数据记录，包含特征和（可选的）标签
  • 训练集 / 验证集 / 测试集：分别用于训练模型、调参、最终评估
  • 损失函数（Loss Function）：衡量模型预测值与真实值之间的差距
  • 梯度下降（Gradient Descent）：通过迭代调整参数来最小化损失函数

过拟合 vs 欠拟合

  • 过拟合：模型在训练集上表现极好，但在测试集上表现差 → 泛化能力弱
  • 欠拟合：模型在训练集和测试集上都表现差 → 学习能力不足
  • 泛化能力：模型对未见过的数据也能做出准确预测的能力
通用学习范式：y = f(X; θ)   →   最小化 Loss(y_true, y_pred)   →   找到最优参数 θ*

机器学习完整流程

  • ① 获取数据：第三方数据、公司业务数据、服务器日志数据等
  • ② 数据预处理：异常值处理（IQR/3σ）、缺失值填充（均值/中位数/插值）
  • ③ 切割训练集和测试集：train_test_split() 通常按 7:2:1 划分训练/验证/测试
  • ④ 特征工程：标准化、归一化、独热编码、分箱、特征交叉等
  • ⑤ 模型选择：根据任务类型（分类/回归/聚类）和数据特点选择合适的算法
  • ⑥ 模型训练：fit() 训练模型，fit_transform() 训练并转换
  • ⑦ 模型预测：predict() 对新数据进行预测
  • ⑧ 模型评估：查看拟合效果 → 欠拟合（训练差+测试差）vs 过拟合（训练好+测试波动）
  • ⑨ 模型部署上线：将训练好的模型应用到生产环境

奥卡姆剃刀原则

  • 如果两个模型泛化表现相同，优先选择简单的模型
  • 泛化能力 = 训练好的模型在未知数据上的表现能力

线性代数

线性代数是机器学习的语言——数据表示为矩阵，模型参数表示为向量，运算本质是线性变换。

核心概念

  • 标量 / 向量 / 矩阵 / 张量：数据的四种表示层级（0D → 1D → 2D → nD）
  • 矩阵乘法：神经网络每一层的前向传播本质：Y = WX + b
  • 转置、逆矩阵：用于求解线性方程组和最小二乘法
  • 特征值 / 特征向量：PCA 降维的核心数学工具
  • 范数：L1 范数（稀疏解）、L2 范数（平滑解），用于正则化
线性模型的一般形式：ŷ = X · w + b
其中 X ∈ Rn×d（样本矩阵），w ∈ Rd（权重向量）

ML 常用的数学概念

  • 导数：描述函数在某点的瞬时变化率，梯度下降的基础
  • 偏导：多元函数对某一变量的导数（其他变量视为常数）
  • 标量：只有大小、没有方向的量（单个数值）
  • 向量：既有大小、又有方向的量（一维数组）
  • 矩阵：二维数组（m×n）。矩阵乘法前提：A 的列 = B 的行，结果矩阵为 m×p
  • 张量：多维数组的统称（n 维），PyTorch 的核心数据结构
  • 梯度：由所有偏导数组成的向量，指向函数增长最快的方向

概率论与统计

机器学习处理的是不确定性问题——概率论提供了在噪声数据中进行推理的数学框架。

核心概念

  • 条件概率与贝叶斯公式：朴素贝叶斯分类器的理论基础
  • 最大似然估计（MLE）：逻辑回归等模型参数估计的常用方法
  • 期望与方差：刻画随机变量分布的集中趋势和离散程度
  • 高斯分布（正态分布）：自然界最常见分布，许多 ML 假设的基础
  • 协方差与相关系数：衡量特征间的线性相关程度
贝叶斯公式：P(A|B) = P(B|A) × P(A) / P(B)
最大似然估计：θMLE = arg maxθ ∏ P(xi | θ)

最优化方法

机器学习的训练过程本质上是一个优化问题——寻找使损失函数最小的模型参数。

梯度下降及其变体

  • 批量梯度下降（BGD）：用全部样本计算梯度，精确但计算量大
  • 随机梯度下降（SGD）：每次随机用1个样本，速度快但波动大
  • 小批量梯度下降（Mini-batch SGD）：折中方案，深度学习中最常用

高级优化器

  • Momentum：引入动量项，加速收敛、减少震荡
  • Adam：结合动量和自适应学习率，目前最流行的优化器
  • 学习率调度：StepLR、CosineAnnealing 等策略动态调整学习率
SGD 更新规则：θt+1 = θt − η ∇L(θt)
其中 η 为学习率，∇L 为损失函数对参数的梯度

信息论基础

信息论为理解模型的不确定性、特征选择以及损失函数的设计提供了理论依据。

核心概念

  • 熵（Entropy）：衡量随机变量的不确定性，熵越大越不确定
  • 交叉熵（Cross-Entropy）：分类任务中最常用的损失函数
  • KL 散度：衡量两个概率分布之间的差异（非对称）
  • 互信息：衡量两个变量之间的相互依赖程度，可用于特征选择
熵：H(X) = −∑ P(x) · log P(x)
交叉熵（二分类）：L = −[y·log(ŷ) + (1−y)·log(1−ŷ)]

数据清洗

真实数据往往是"脏"的——缺失值、异常值、重复数据等问题需要在建模前处理。

处理方法

  • 缺失值处理：删除（缺失过多时）、均值/中位数填充、KNN 填充、插值法
  • 异常值检测：3σ 原则 · 箱线图 IQR · 孤立森林
  • 异常值处理：截断（capping）、变换（log 变换）、删除
  • 重复值：drop_duplicates() 直接删除
  • 数据类型转换：确保数值列是 int/float，类别列是 category/object
Python 
import pandas as pd

# 缺失值检查与填充
df.isnull().sum()
df['age'].fillna(df['age'].median(), inplace=True)

# IQR 异常值检测
Q1 = df['price'].quantile(0.25)
Q3 = df['price'].quantile(0.75)
IQR = Q3 - Q1
outliers = (df['price'] < Q1 - 1.5*IQR) | (df['price'] > Q3 + 1.5*IQR)

特征工程

"数据和特征决定了机器学习的上限，模型和算法只是逼近这个上限。"特征工程是 ML 中最关键的环节之一。

特征处理流水线

  • 标准化（Standardization）：使数据均值为 0、标准差为 1，适用于 SVM、PCA 等
  • 归一化（Normalization）：将数据缩放到 [0,1] 区间，适用于神经网络
  • 独热编码（One-Hot Encoding）：将类别特征转化为二进制向量
  • 分箱（Binning）：将连续值离散化，增强模型非线性表达能力
  • 多项式特征：创建特征的高阶组合，捕捉非线性关系
  • 特征交叉：组合多个特征创造新特征

标准化 vs 归一化

  • 归一化（MinMaxScaler）：受异常值影响大，适合数据分布已知的场景from sklearn.preprocessing import MinMaxScaler
  • 标准化（StandardScaler）：受异常值影响小，适合数据分布未知的场景，更常用from sklearn.preprocessing import StandardScaler
Python 
from sklearn.preprocessing import StandardScaler, OneHotEncoder

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

encoder = OneHotEncoder(sparse_output=False)
X_encoded = encoder.fit_transform(df[['category']])

数据划分与采样

合理的数据划分是模型评估可靠性的基础，划分不当会导致评估结果失真。

划分策略

  • 留出法（Hold-out）：简单按比例划分（如 7:2:1），最常用
  • 分层抽样（Stratified Split）：保持各类别比例一致，适用于不平衡数据
  • 交叉验证（Cross-Validation）：K 折 CV 更充分利用数据
  • 时间序列划分：按时间顺序划分，防止未来信息泄露

不平衡数据处理

  • 过采样（Oversampling）：SMOTE 合成少数类样本
  • 欠采样（Undersampling）：随机删除多数类样本
  • 代价敏感学习：给少数类错分更高的惩罚权重
Python 
from sklearn.model_selection import train_test_split
from imblearn.over_sampling import SMOTE

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2, stratify=y, random_state=42
)

smote = SMOTE(random_state=42)
X_resampled, y_resampled = smote.fit_resample(X_train, y_train)

监督学习

监督学习是机器学习中最成熟的领域——给定带标签的训练数据 (X, y)，学习从输入到输出的映射关系 f: X → y。

核心思想

通过最小化预测值 f(X) 与真实标签 y 之间的损失，找到最优映射函数。

两大任务类型

  • 回归（Regression）：预测连续值。如房价预测、股票价格预测
  • 分类（Classification）：预测离散类别。如垃圾邮件识别、图像识别
算法任务核心思想优点缺点
线性回归回归最小二乘法简单可解释只能拟合线性关系
逻辑回归二分类Sigmoid + 最大似然输出概率，可解释线性决策边界
决策树分类/回归信息增益递归划分可视化、无需归一化易过拟合
SVM分类/回归最大间隔超平面高维有效大规模数据慢
KNN分类/回归距离投票无需训练、简单计算量大

线性回归

最基础也是最重要的回归算法——假设目标值与特征之间存在线性关系。

ŷ = w0 + w1x1 + w2x2 + ... + wnxn = Xw

三个关键假设

  • 线性性：自变量与因变量之间存在线性关系
  • 独立性：各观测值相互独立
  • 同方差性：误差项的方差恒定
  • 正态性：误差项服从正态分布

求解方法对比

  • 正规方程（解析解）：w = (XᵀX)⁻¹Xᵀy，一步到位直接计算，计算量大只适合小数据，可能无逆矩阵导致无解
  • 梯度下降（迭代解）：通过迭代更新参数 wnew = wold − lr × ∇L，逐步逼近最优解，适合大数据

梯度下降的三种形态

  • 全梯度下降（BGD/FGD）：用全部样本计算梯度，精确但计算慢
  • 随机梯度下降（SGD）：每次随机用 1 个样本，速度快但波动大from sklearn.linear_model import SGDRegressor
  • 小批量梯度下降（MBGD）：折中方案，每次用一批样本，深度学习最常用

正则化变体

  • Ridge 回归（L2）：损失 + λ‖w‖₂²，收缩系数但不为零
  • Lasso 回归（L1）：损失 + λ‖w‖₁，可产生稀疏解（特征选择）
  • ElasticNet：L1 + L2 结合，兼具两者优点
Python 
from sklearn.linear_model import LinearRegression, Ridge, Lasso

model = LinearRegression()
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
print('R²:', model.score(X_test, y_test))

# 梯度下降方式求解
from sklearn.linear_model import SGDRegressor
sgd = SGDRegressor(max_iter=1000, learning_rate='constant', eta0=0.01)
sgd.fit(X_train, y_train)

逻辑回归

名为"回归"，实为分类——通过 Sigmoid 函数将线性输出映射到 [0,1] 概率区间。

P(y=1|x) = σ(z) = 1 / (1 + e−z)，其中 z = Xw
σ(z) ∈ (0, 1)，可解释为样本属于正类的概率

核心要点

  • 损失函数：交叉熵损失（Cross-Entropy），而非 MSE
  • 决策边界：P=0.5 对应的超平面，是线性的
  • 多分类扩展：Softmax 回归（OvR 或 Multinomial）
  • 概率输出：天然输出概率值，便于设置不同阈值

底层优化原理

  • 将求最大概率的似然函数取负对数
  • 得到最小化的交叉熵损失函数
  • 更简单、稳定，适配 AI 通用优化方法：梯度下降
Python 
from sklearn.linear_model import LogisticRegression

model = LogisticRegression(penalty='l2', C=1.0, solver='lbfgs')
model.fit(X_train, y_train)
y_prob = model.predict_proba(X_test)[:, 1]  # 获取概率

决策树

模拟人类决策过程的树形结构模型，通过递归划分特征空间进行分类或回归。

分裂准则

  • 信息增益（ID3）：选择使信息熵减少最多的特征进行分裂
  • 信息增益率（C4.5）：克服 ID3 偏向取值多的特征
  • 基尼指数（CART）：衡量节点纯度，值越小越纯，计算更快
熵：H(D) = −∑ pk · log₂(pk)
基尼指数：Gini(D) = 1 − ∑ pk²
信息增益：IG(D, A) = H(D) − H(D|A)

剪枝策略

  • 预剪枝：限制最大深度、最小样本分裂数、最小叶节点样本数
  • 后剪枝：先完整生长，再从底向上剪除不重要的分支
  • 代价复杂度剪枝（CCP）：CART 使用的剪枝方法
Python 
from sklearn.tree import DecisionTreeClassifier

model = DecisionTreeClassifier(
    max_depth=5, min_samples_split=10, criterion='gini'
)
model.fit(X_train, y_train)

支持向量机 SVM

SVM 寻找一个最优超平面，使不同类别的样本到超平面的最小距离（间隔）最大化。

核心思想

最大间隔分类 + 核技巧 = SVM。支持向量是离决策边界最近的那些样本，它们是定义边界的关键。

三大核心概念

  • 最大间隔：不仅正确分类，还要让分类边界尽可能远离各类样本
  • 软间隔（C 参数）：允许少量错分，C 越大惩罚越重（越容易过拟合）
  • 核技巧（Kernel Trick）：隐式将数据映射到高维空间，使非线性可分变为线性可分
核函数公式适用场景
线性核K(x,y) = x·y线性可分、特征维度高
多项式核K(x,y) = (x·y + c)ᵈ有一定非线性
RBF 核K(x,y) = exp(−γ‖x−y‖²)最常用、非线性场景

K近邻 KNN

"近朱者赤，近墨者黑"——一个样本的类别由其 K 个最近邻居的类别投票决定。

特点

KNN 属于懒惰学习（Lazy Learning）——没有显式训练过程，所有计算都在预测时进行。

关键要素

  • K 值选择：K 太小→过拟合/噪声敏感；K 太大→欠拟合/边界模糊。通常用交叉验证选择
  • 距离度量：欧氏距离（最常用）、曼哈顿距离、余弦相似度
  • 投票机制：多数投票（分类）、加权平均（回归）
  • 必须标准化：KNN 对特征尺度敏感，预处理时必须做标准化
欧氏距离：d(x, y) = √(∑(xi − yi)²)
分类规则：ŷ = mode(yneighbor₁, ..., yneighborₖ)

分类 vs 回归

  • KNN 分类：距离最近的 K 个数据投票决定类别from sklearn.neighbors import KNeighborsClassifier
  • KNN 回归：距离最近的 K 个数据的平均值作为预测结果from sklearn.neighbors import KNeighborsRegressor
Python 
from sklearn.neighbors import KNeighborsClassifier
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import train_test_split

# 必须先标准化
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2)

model = KNeighborsClassifier(n_neighbors=5)
model.fit(X_train, y_train)
print(model.score(X_test, y_test))

朴素贝叶斯

基于贝叶斯定理的分类器，其"朴素"之处在于假设所有特征相互独立。

P(C|X) = P(X|C) × P(C) / P(X) ∝ P(X|C) × P(C)
其中 P(X|C) = ∏ P(xi|C)（朴素条件独立假设）

三种变体

  • 高斯朴素贝叶斯：假设特征服从高斯分布（连续特征）
  • 多项式朴素贝叶斯：适用于离散计数特征（如文本词频）
  • 伯努利朴素贝叶斯：适用于二值特征（出现/不出现）

优缺点

  • 优点：计算速度快、小数据表现好、适合文本分类（如垃圾邮件过滤）
  • 缺点：特征独立假设在实际中很难满足

无监督学习

无监督学习在无标签数据中发现隐藏的结构——让数据"自己说话"。

核心思想

不需要人工标注，直接从数据中学习模式，适用于标签获取成本高的场景。

三大任务方向

  • 聚类（Clustering）：将相似样本归为一组 → K-Means、DBSCAN、层次聚类
  • 降维（Dimensionality Reduction）：减少特征数量同时保留主要信息 → PCA、t-SNE
  • 关联规则（Association Rules）：发现特征间的关联关系 → Apriori、FP-Growth

K-Means 聚类

最经典的聚类算法，通过迭代优化使簇内距离平方和最小。

算法步骤

  • ① 随机选择 K 个初始聚类中心
  • ② 将每个样本分配到最近的聚类中心
  • ③ 重新计算每个簇的均值作为新的聚类中心
  • ④ 重复 ②③ 直到中心不再变化或达到最大迭代次数
目标函数：J = ∑ ∑ ‖x − μi‖²（每个簇内的样本到质心的距离平方和）

关键问题

  • K 值选择：肘部法则（Elbow Method）、轮廓系数（Silhouette Score）
  • 初始点敏感：K-Means++ 优化初始中心点选择
  • 仅适合球形簇：对非球形分布的数据效果差
Python 
from sklearn.cluster import KMeans

model = KMeans(n_clusters=3, init='k-means++', random_state=42)
labels = model.fit_predict(X)
print('惯性值:', model.inertia_)

DBSCAN

基于密度的聚类算法，能够发现任意形状的簇，同时自动识别噪声点。

核心概念

  • ε 邻域：以某点为中心、ε 为半径的区域
  • 核心点（Core Point）：邻域内样本数 ≥ minPts 的点
  • 边界点（Border Point）：邻域内样本数 < minPts，但在核心点的邻域内
  • 噪声点（Noise Point）：既不是核心点也不是边界点

vs K-Means

  • 优点：不需要预设 K 值、能发现任意形状簇、自动识别噪声
  • 缺点：参数 ε 和 minPts 难调、密度差异大时效果差、高维数据效果差
Python 
from sklearn.cluster import DBSCAN

model = DBSCAN(eps=0.5, min_samples=5)
labels = model.fit_predict(X)
print('噪声点数量:', sum(labels == -1))

PCA 主成分分析

最常用的线性降维方法——将数据投影到方差最大的方向（主成分），用较少维度保留最多信息。

算法步骤

  • ① 数据中心化（减去均值）
  • ② 计算协方差矩阵
  • ③ 对协方差矩阵做特征值分解
  • ④ 按特征值降序排列，选取前 k 个特征向量作为主成分
  • ⑤ 将原始数据投影到选定的主成分上
协方差矩阵：C = XᵀX / n
方差解释率：λi / ∑λj（第 i 个主成分解释的方差比例）

应用场景

  • 数据可视化：降至 2D/3D 便于绘图
  • 去噪：丢弃小特征值对应的成分（往往对应噪声）
  • 加速训练：减少特征维度降低计算量

层次聚类

通过构建层次树（树状图/Dendrogram）来展示样本之间的嵌套关系，不需要预设簇数量。

两种策略

  • 自底向上（凝聚法 Agglomerative）：每个样本起始为一个簇，逐步合并最相似的簇
  • 自顶向下（分裂法 Divisive）：所有样本起始为一个大簇，逐步分裂

簇间距离度量

  • 单链接（Single Linkage）：取两个簇最近样本距离 → 对噪声敏感
  • 全链接（Complete Linkage）：取两个簇最远样本距离 → 倾向球形簇
  • 平均链接（Average Linkage）：取所有样本对距离平均值 → 折中方案
  • Ward 法：最小化合并后的方差增量 → 最常用

集成学习

"三个臭皮匠，顶个诸葛亮"——集成学习通过组合多个弱学习器来构建一个强学习器。

核心思想

个体学习器要"好而不同"——每个基学习器要有一定准确度，同时彼此之间要有差异。

方法代表算法基学习器关系核心思想
Bagging随机森林并行、独立Bootstrap 采样 + 投票/平均
BoostingAdaBoost / GBDT / XGBoost串行、依赖逐步修正前一个学习器的错误
StackingStacked Generalization分层用元学习器组合多个基学习器

Bagging

Bootstrap Aggregating——通过自助采样生成多个训练集，训练多个基学习器，最后投票或取平均。

核心机制

  • Bootstrap 采样：从原始训练集中有放回地随机抽取 n 个样本，重复 m 次得到 m 个训练集
  • 并行训练：m 个基学习器独立训练，互不影响
  • 聚合预测：分类→多数投票；回归→简单平均
  • 袋外估计（OOB）：未被采样的约 36.8% 样本可作为验证集
为什么有效？

Bagging 主要降低方差，适合高方差模型（如决策树）。每个基学习器在不同的数据子集上训练，平均后能显著减少过拟合。

随机森林

Bagging + 决策树的经典组合——在 Bagging 基础上增加了特征的随机选择，进一步降低树之间的相关性。

双重随机性

  • 样本随机：每个树用 Bootstrap 采样的不同训练集训练
  • 特征随机：每个节点分裂时，只从随机选取的 k 个特征中选择最优分裂特征
建议 k = log₂(d)（分类）  或  k = d/3（回归）
其中 d 为总特征数

优势

  • 抗过拟合能力强（双重随机性）
  • 可输出特征重要性排名
  • 能处理高维数据，不需要特征选择
  • 训练速度快，天然支持并行
Python 
from sklearn.ensemble import RandomForestClassifier

model = RandomForestClassifier(n_estimators=100, max_depth=10)
model.fit(X_train, y_train)
# 特征重要性
print(model.feature_importances_)

AdaBoost

自适应增强——通过调整样本权重让后续学习器更关注前一个学习器分错的样本。

算法流程

  • ① 初始化所有样本权重相等：wi = 1/n
  • ② 训练一个基学习器，计算其加权错误率
  • ③ 根据错误率计算该学习器的权重 α（错误率越低权重越大）
  • ④ 增大错分样本的权重，减小正确分类样本的权重
  • ⑤ 重复 ②-④，最终加权投票得到强学习器
学习器权重：α = 0.5 × ln((1−ε) / ε)
其中 ε 为加权错误率，α 衡量该学习器的"话语权"

特点

  • 优点：简单、不易过拟合、自动做特征选择
  • 缺点：对噪声和异常值敏感（会过度关注错分的噪声样本）

GBDT

梯度提升决策树——不同于 AdaBoost 调整样本权重，GBDT 每步拟合的是前一步的残差（负梯度）。

核心思想

每一棵新树的目标不是直接预测 y，而是预测当前模型残差的负梯度方向，逐步逼近真实值。

Fm(x) = Fm−1(x) + ν · hm(x)
其中 ν 为学习率（shrinkage），hm 拟合的是残差 r = y − Fm−1(x)

关键参数

  • n_estimators：树的数量，越大越容易过拟合
  • learning_rate（shrinkage）：每棵树的贡献缩放因子，小值需配合更多树
  • max_depth：通常较浅（3-5），保持基学习器是弱学习器
  • subsample：随机采样比例，<1 可以增加随机性（SGBT）

XGBoost

GBDT 的工程优化版——在算法层面和系统层面都做了大量优化，Kaggle 竞赛中的常胜将军。

相比 GBDT 的改进

  • 二阶泰勒展开：利用二阶导数信息，收敛更精确
  • 显式正则项：目标函数中加入叶子节点数和 L2 正则项
  • 列抽样（Column Subsampling）：类似随机森林，减少过拟合
  • Shrinkage：缩减每棵树的贡献，防止过拟合
  • 缺失值处理：自动学习缺失值的分裂方向
  • 系统优化：并行化、缓存优化、稀疏感知、分位点近似分割
Python 
from xgboost import XGBClassifier

model = XGBClassifier(
    n_estimators=100, learning_rate=0.1, max_depth=5,
    subsample=0.8, colsample_bytree=0.8
)
model.fit(X_train, y_train, eval_set=[(X_test, y_test)])

Stacking

堆叠泛化——用元学习器（Meta-Learner）来学习如何最优组合多个一级学习器的预测结果。

两层架构

  • 第一层（基学习器）：多个不同类型的模型（如 SVM、RF、XGBoost），产生各自的预测结果
  • 第二层（元学习器）：以第一层模型的输出作为特征，学习最优组合方式
  • 关键技巧：使用交叉验证避免数据泄露（用验证集的预测来训练元学习器）
Python 
from sklearn.ensemble import StackingClassifier
from sklearn.linear_model import LogisticRegression

base_models = [
    ('rf', RandomForestClassifier()),
    ('svm', SVC(probability=True)),
    ('xgb', XGBClassifier()),
]
model = StackingClassifier(
    estimators=base_models,
    final_estimator=LogisticRegression(),
    cv=5

model.fit(X_train, y_train)

感知机

神经网络的起源——最基础的二分类线性模型，是理解深度学习的起点。

y = sign(w·x + b) = sign(∑ wixi + b)
输入加权求和后经过阶跃激活函数输出 ±1

核心要点

  • 学习规则：w ← w + η · ytrue · x（仅在分类错误时更新）
  • 局限性：只能解决线性可分问题，连 XOR 都无法解决（由此导致了第一次 AI 寒冬）
  • 历史意义：感知机的局限性催生了多层网络（MLP）

多层感知机 MLP

在输入和输出之间增加隐藏层，并使用非线性激活函数，从而能够拟合任意复杂函数。

关键组件

  • 前向传播：逐层计算 a(l+1) = σ(W(l)a(l) + b(l))
  • 反向传播（Backpropagation）：通过链式法则逐层计算梯度
  • 激活函数：ReLU（主流）、Sigmoid、Tanh、LeakyReLU
  • 万能逼近定理：一个具有足够神经元和适当激活函数的单隐藏层就可以逼近任意连续函数
ReLU(x) = max(0, x)   |   Sigmoid(x) = 1/(1+e−x)
反向传播核心链式法则：∂L/∂w = ∂L/∂a · ∂a/∂z · ∂z/∂w
激活函数值域导数范围适用场景注意事项
Sigmoid(0, 1)(0, 0.25]二分类输出层、浅隐藏层非0中心化，梯度消失严重
Tanh(-1, 1)(0, 1]浅隐藏层0 中心化，梯度消失有所缓解
ReLU[0, +∞)0 或 1深隐藏层（主流）负样本直接忽略，缓解过拟合
Softmax概率和=1—多分类输出层所有输出概率总和为 1
激活函数选择口诀

浅隐藏层 Tanh · 深隐藏层 ReLU · 二分类输出 Sigmoid · 多分类输出 Softmax

参数初始化方法

  • 均匀/正态随机初始化：uniform_()normal_() 基础方法
  • Kaiming/He 初始化：kaiming_uniform_() 只考虑输入节点，适用于 ReLU 及其变体
  • Xavier 初始化：xavier_uniform_() 同时考虑输入和输出，适用于 Sigmoid/Tanh

五大损失函数

  • 多分类交叉熵：nn.CrossEntropyLoss() 底层自动 Softmax + 损失计算
  • 二分类交叉熵：nn.BCELoss() 需先手动 sigmoid 转概率
  • MSE 均方误差：nn.MSELoss() 对异常值敏感，放大误差，用于回归
  • MAE 平均绝对误差：nn.L1Loss() 对异常值鲁棒，用于回归
  • Smooth L1：nn.SmoothL1Loss() MSE 和 MAE 的结合，[-1,1] 内平缓，之外线性
PyTorch 
import torch.nn as nn

# 定义神经网络（1继承 + 2重写）
class MyNet(nn.Module):
    def __init__(self):
        super().__init__()  # 必须调用父类 init
        self.fc1 = nn.Linear(784, 128)
        self.fc2 = nn.Linear(128, 10)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)  # 多分类输出层不加 softmax
        return x

model = MyNet()
criterion = nn.CrossEntropyLoss()  # 底层自动 softmax
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)

PyTorch 框架基础

PyTorch 是深度学习最主流的框架——数据以张量（Tensor）形式存储，提供自动微分和丰富的神经网络模块。

NumPy vs Tensor 分工

NumPy 处理原始数据和结果后处理（CPU），PyTorch Tensor 负责模型训练和推理（GPU），两者通过 .numpy() 和 torch.from_numpy() 互相转换。

创建张量

  • 基本创建：torch.tensor(data, dtype=)torch.Tensor(data)
  • 线性张量：torch.arange(start, end, step)torch.linspace(start, end, steps)
  • 随机张量：torch.rand(size) 0~1均匀 · torch.randn(size) 正态分布 · torch.randint(low, high, size)
  • 全0/1/指定值：torch.zeros(size)torch.ones(size)torch.full(size, value)

张量类型转换（推荐方式）

  • 转整数：tensor.byte()tensor.short()tensor.int()tensor.long()
  • 转浮点：tensor.half()tensor.float()tensor.double()
  • 通用转换：tensor.to(dtype) 或 tensor.type(dtype)

Tensor ↔ NumPy 互转

  • Tensor → NumPy：tensor.numpy()（共享内存）或 tensor.numpy().copy()（独立）
  • NumPy → Tensor：torch.from_numpy(arr)（共享内存）或 torch.tensor(arr)（独立）
  • 注意：开启自动微分的张量需先 .detach() 剥离后才能转 NumPy

自动微分（Autograd）

  • 开启求导：创建张量时设置 requires_grad=True
  • 反向传播：loss.backward() 自动计算梯度并存储到 .grad 属性
  • 权重更新：Wnew = Wold − lr × W.grad
  • 梯度清零：循环训练时每次反向传播前需 optimizer.zero_grad()

卷积神经网络 CNN

专门为处理网格状数据（如图像）设计的网络结构，通过局部连接和权重共享大幅减少参数量。

图像基础

  • 图像表示：NumPy → (H, W, C) · PyTorch → (C, H, W)
  • 通道（Channel）：灰度图 1 通道、RGB 3 通道，像素值范围 [0, 255]
  • 本质：图像是 1 个或 3 个二维矩阵，矩阵的值就是像素点（特征值）

卷积层详解

  • 卷积核（滤波器）：带有共享参数的神经元，每个核提取一个二维特征图
  • 卷积计算：卷积核与输入区域做点乘，结果 = 新特征图的一个像素值
  • 多通道计算：RGB 3 通道分别点乘后求和，得到 1 个特征值
  • 多卷积核：N 个卷积核 → 提取 N 个不同的二维特征图
特征图尺寸：N = ⌊(W − F + 2P) / S⌋ + 1
W=输入尺寸 · F=卷积核大小 · P=补零圈数 · S=步长

Padding 与 Stride

  • Padding（补零）：防止边缘信息丢失（边缘像素只被计算少数次），保持特征图尺寸不变
  • Stride（步长）：窗口每次移动的像素数，增大步长 = 降维 + 增大感受野

池化层（Pooling）

  • 作用：降维（只在 H、W 维度），无卷积核参与，只有池化窗口
  • 最大池化：取窗口内最大值，保留显著特征nn.MaxPool2d(kernel_size, stride, padding)
  • 平均池化：取窗口内平均值，平滑特征nn.AvgPool2d(kernel_size, stride, padding)
  • 多通道池化：不改变通道数，每个通道独立池化
PyTorch 
import torch.nn as nn

# Conv2d(输入通道, 输出通道(卷积核数), 核大小, 步长, 填充)
conv = nn.Conv2d(in_channels=3, out_channels=32, kernel_size=3, stride=1, padding=1)
pool = nn.MaxPool2d(kernel_size=2, stride=2)

model = nn.Sequential(
    nn.Conv2d(3, 32, kernel_size=3, padding=1),
    nn.ReLU(),

)

RNN / LSTM

循环神经网络专为序列数据设计——当前输出不仅依赖当前输入，还依赖之前的"记忆"。

RNN 的问题

  • 梯度消失：长序列中早期时间步的梯度趋于零，难以学习长距离依赖
  • 梯度爆炸：梯度指数增长，导致训练不稳定（可用梯度裁剪解决）

LSTM 的解决方案：门控机制

  • 遗忘门（Forget Gate）：决定丢弃哪些旧信息
  • 输入门（Input Gate）：决定存储哪些新信息
  • 输出门（Output Gate）：决定输出哪些信息
  • 细胞状态（Cell State）：贯穿整个序列的"高速公路"，梯度可以无损传播

词嵌入层（Embedding）

  • 作用：将词用低维稠密向量表示，保留上下文语义，输入到 RNN 层
  • 流程：文本 → jieba 分词 → 词 → 下标 → 向量
  • API：nn.Embedding(num_embeddings, embedding_dim)
  • num_embeddings：词汇表大小 · embedding_dim：每个词的向量维度

RNN 层 API

  • 创建：rnn = nn.RNN(input_size, hidden_size, num_layers)
  • input_size：输入特征维度（一般设为词向量维度）
  • hidden_size：隐藏层 h 的维度，也是当前层神经元的输出维度
  • 调用：output, h_t = rnn(x, h_{t-1})— x 和 output 形状为 (序列长度, batch, 词向量维度)
应用场景

文本生成、机器翻译、语音识别、时间序列预测等一切序列相关任务。不过在 NLP 领域已逐渐被 Transformer 取代。

Transformer

2017年提出的革命性架构，用自注意力机制取代循环结构，成为当前大模型（GPT、BERT 等）的基石。

核心突破

自注意力机制（Self-Attention）让每个位置都能直接关注序列中的任意位置，解决了 RNN 的长距离依赖问题，同时支持并行训练。

核心组件

  • 自注意力（Self-Attention）：Q（查询）、K（键）、V（值）三矩阵计算注意力权重
  • 多头注意力（Multi-Head Attention）：并行运行多组注意力，捕捉不同类型的关系
  • 位置编码（Positional Encoding）：为模型注入序列顺序信息
  • 前馈网络（FFN）：每个位置独立应用的全连接层
  • 残差连接 + LayerNorm：稳定深层网络训练
Attention(Q,K,V) = softmax(QKᵀ / √dk) · V
缩放因子 √dk 防止点积过大导致 Softmax 梯度消失

经典变体

  • BERT：仅用 Encoder，双向理解 → 分类/理解任务
  • GPT：仅用 Decoder，自回归生成 → 文本生成任务
  • ViT：将图像分割为 Patch 序列，用 Transformer 处理视觉任务

评估指标

选择合适的评估指标是模型开发的关键——不同场景有不同的侧重点。

指标公式适用场景
准确率(TP+TN)/(TP+TN+FP+FN)类别平衡场景
精确率TP/(TP+FP)关注误报（垃圾邮件 → 不能误判正常邮件）
召回率TP/(TP+FN)关注漏报（癌症筛查 → 不能漏诊）
F1-Score2×P×R/(P+R)精确率与召回率的调和平均
AUC-ROCROC 曲线下面积综合评估分类器排序能力

混淆矩阵

  • TP（True Positive）：正类预测为正类 ✓
  • FP（False Positive）：负类预测为正类 ✗（误报/假阳性）
  • FN（False Negative）：正类预测为负类 ✗（漏报/假阴性）
  • TN（True Negative）：负类预测为负类 ✓

回归指标

  • MSE（均方误差）：对异常值敏感（平方放大误差），任意位置都可导但 loss 越大梯度越大，易梯度爆炸
  • MAE（平均绝对误差）：对异常值鲁棒（线性误差），导数 ±1 或 0 点不可导，不容易梯度爆炸但可能跳过极小值
  • R²（决定系数）：模型解释的方差比例，越接近 1 越好
Python 
from sklearn.metrics import (accuracy_score, precision_score,
    recall_score, f1_score, confusion_matrix, classification_report)

# 先预测再评估
y_pred = model.predict(X_test)
print('准确率:', accuracy_score(y_test, y_pred))
print('精确率:', precision_score(y_test, y_pred, pos_label=4))
print('召回率:', recall_score(y_test, y_pred, pos_label=4))
print('F1:', f1_score(y_test, y_pred, pos_label=4))
print('混淆矩阵:\n', confusion_matrix(y_test, y_pred))
print('分类报告:\n', classification_report(y_test, y_pred))

交叉验证

交叉验证是评估模型泛化能力的黄金标准——充分利用有限的数据进行可靠的评估。

常用方法

  • K 折交叉验证：将数据分为 K 份，每次用 1 份验证、K-1 份训练，重复 K 次取平均
  • 留一交叉验证（LOOCV）：K=n 的特殊情况，最充分利用数据但计算量大
  • 分层 K 折：每折保持类别比例，适用于不平衡数据
  • 时间序列 CV：按时间顺序划分，保证训练数据都在测试数据之前
Python 
from sklearn.model_selection import cross_val_score

scores = cross_val_score(model, X, y, cv=5, scoring='accuracy')
print(f'CV Mean: {scores.mean():.4f} ± {scores.std():.4f}')

超参数调优

超参数是模型训练前设定的参数（不像权重从数据中学习），合理的调优能显著提升模型性能。

三种主流方法

  • 网格搜索（Grid Search）：遍历所有参数组合，全面但计算量随参数指数增长
  • 随机搜索（Random Search）：随机采样参数组合，效率更高、更易发现优质区域
  • 贝叶斯优化：利用历史评估结果指导下一次采样，效率最高。代表：Optuna、Hyperopt
Python 
from sklearn.model_selection import GridSearchCV

params = {'n_estimators': [100, 200], 'max_depth': [5, 10, None]}
search = GridSearchCV(RandomForestClassifier(), params, cv=5)
search.fit(X_train, y_train)
print(search.best_params_, search.best_score_)

正则化

正则化通过在损失函数中添加惩罚项来限制模型复杂度，是防止过拟合的核心手段。

L1 正则化（Lasso）

  • 惩罚项：λ ∑|wi|
  • 效果：产生稀疏解——部分权重变为 0，天然做特征选择
  • 几何解释：约束区域是菱形，最优解容易落在坐标轴上

L2 正则化（Ridge）

  • 惩罚项：λ ∑wi²
  • 效果：权重整体缩小但不为零，分布更平滑
  • 几何解释：约束区域是圆形，最优解均匀分布

深度学习的正则化方法

  • Dropout：训练时随机丢弃一部分神经元，相当于每次训练一个子网络
  • Early Stopping：验证集误差不再下降时提前停止训练
  • Batch Normalization：归一化每层输入，本身有一定的正则化效果
  • Data Augmentation：通过对训练数据做变换（旋转、裁剪、加噪）增加数据多样性
L1：Loss(w) = Loss0 + λ‖w‖₁
L2：Loss(w) = Loss0 + λ‖w‖₂²

偏差与方差

偏差-方差权衡是理解模型泛化性能的核心框架——所有模型的误差都可分解为偏差、方差和噪声。

Etotal = Bias² + Variance + Noise
泛化误差 = 偏差²（模型平均预测与真实值的差距）+ 方差（模型预测的波动程度）+ 不可约误差（噪声）
问题表现原因解决方案
高偏差（欠拟合）训练误差高、测试误差高模型太简单增加模型复杂度、增加特征、减少正则化
高方差（过拟合）训练误差低、测试误差高模型太复杂增加数据、增加正则化、降低模型复杂度
直觉理解

偏差 → 瞄准是否准确；方差 → 多次射击的一致性。好的模型需要既瞄得准（低偏差）又打得好（低方差）。
